# 🌿 E-Plant Shopping Application

## 📌 Project Description

The **E-Plant Shopping Application** is a modern web application built using **React (Vite) and Redux Toolkit**. It allows users to browse a variety of plants, filter them by categories, and manage a shopping cart with dynamic updates.

---

## 🚀 Features

* 🌱 Browse plant products
* 📂 Filter plants by category
* 🛒 Add items to cart
* ➕ Increase / ➖ decrease quantity
* ❌ Remove items from cart
* 💰 Automatic total price calculation
* 📄 About Us page
* ⚡ Fast performance using Vite
* 🧠 State management using Redux Toolkit

---

## 🛠️ Technologies Used

* React.js (with Vite)
* Redux Toolkit
* JavaScript (ES6+)
* HTML5 & CSS3

## ⚙️ Installation & Setup

1. Clone the repository:

```id="cmd01"
git clone https://github.com/ibm-developer-skills-network/e-plantShopping.git
```

2. Navigate to the project folder:

```id="cmd02"
cd e-plantShopping
```

3. Install dependencies:

```id="cmd03"
npm install
```

4. Start development server:

```id="cmd04"
npm run dev
```

---

## 🧠 State Management

The application uses **Redux Toolkit**:

* `CartSlice.jsx` → Handles cart logic (add, remove, update quantity)
* `store.js` → Central Redux store
